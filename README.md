# Online Banking System

## Overview

This project implements a simple banking system using Java and MySQL.
It allows users to create accounts, deposit money, withdraw funds, transfer money, and view transaction history.

## Features

* User registration
* Account creation
* Deposit money
* Withdraw money
* Transfer funds
* Balance checking
* Transaction history

## Tech Stack

* Java
* MySQL
* JDBC
* SQL

## Database Tables

* users
* accounts
* transactions

Currently under development.
